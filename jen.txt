A — Common UI actions
Open Jenkins → New Item (left menu)
Enter job name → choose Freestyle project → OK
Configure → Source Code Management: Git → paste repo URL → set Branches to build (*/main or */master)
Build steps → Add build step → Invoke top-level Maven targets → select MAVEN_HOME → set Goals (clean, install, test, etc.)
Post-build → choose actions like Archive the artifacts, Build other projects, Deploy WAR/EAR to a container
Save → Build Now or use pipeline view to run flows
